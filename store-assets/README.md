### Full description

```
img2prompt turns any on-page image — photo, CSS background, canvas, video frame — into three ready-to-paste prompts for the major AI image models.

How it works
1. Hit the shortcut (default Cmd/Ctrl+Shift+Y).
2. Drag to select any region of the page.
3. Pick from three prompt formats in the popup:
   • Midjourney (with --ar / --style raw / --v parameters)
   • Stable Diffusion / Flux (positive + negative + weighting notes)
   • DALL·E / GPT-4o / Nano Banana (natural-language paragraph)
4. Click copy, paste into your favorite tool, iterate.

Bring your own Vision API
img2prompt does not ship a server. You plug in any OpenAI-compatible endpoint
— OpenAI, Azure, a local model, a Chinese cloud — and the request goes
directly from your browser to your chosen provider, using your own API key.
Nothing is sent to us.

Privacy
• No analytics, no telemetry, no backend.
• Your API key lives in chrome.storage.sync (synced only to your own Google account).
• Images stay in memory; they are never written to localStorage or sent anywhere other than your configured endpoint.
• The last 50 generations are cached locally in IndexedDB for quick reuse; you can clear them any time.

Permissions (minimum)
• activeTab — inject the selection overlay on the page you triggered
• storage — remember your API config and history
• scripting — fallback overlay injection on pages that need it

Perfect for designers, visual-research accounts, and anyone building a style
mood board from scrolling Pinterest, Dribbble, Xiaohongshu, or X.
```

### Permission justifications (copy into the Chrome Web Store form)

- `activeTab` — "Needed to inject a transient selection overlay and to call
  `chrome.tabs.captureVisibleTab` on the tab where the user pressed the
  shortcut. Not used to observe browsing."
- `storage` — "Needed to persist the user's API configuration
  (`chrome.storage.sync`) and local-only prompt history (IndexedDB)."
- `scripting` — "Needed as a fallback to inject the selection overlay on pages
  where the content script was not preloaded. Only runs on the tab the user
  explicitly triggers."
